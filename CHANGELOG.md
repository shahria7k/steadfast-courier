# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added

- Initial release of Steadfast Courier SDK
- Full TypeScript support with comprehensive type definitions
- Order management (single and bulk order creation)
- Delivery status checking (by consignment ID, invoice, or tracking code)
- Balance checking
- Return request management
- Payment operations
- Police station data retrieval
- Webhook handling with Express, Fastify, and generic adapters
- Input validation utilities
- Custom error classes
- Comprehensive test suite
- Documentation and examples
